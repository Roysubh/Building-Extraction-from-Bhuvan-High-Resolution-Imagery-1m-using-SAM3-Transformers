# Building-Extraction-from-NRSC-Bhuvan-Imagery-using-SAM3-Transformers-
Prompt-based building footprint extraction from NRSC Bhuvan 1 m resolution satellite imagery using Segment Anything Model v3 (SAM3) with the Transformers backend. The workflow demonstrates a reproducible GeoAI pipeline integrating QGIS and Python for automatic building mask generation.
